# LoadingButton

 一个加载功能的按钮控件，支持按钮收缩，完成和失败的图标文字切换

### 特性
   * 支持按钮收缩
   * 支持加载取消，加载完成和失败图标替换
   * 可设置加载动画颜色、大小、位置

disableClickOnLoading|boolen|设置加载时禁用点击
shrinkDuration|int(default:450)|收缩动画时间
